CSS Notes


CSS Purpose: 
	- Defines styles for web pages
		○ Layout
		○ Design 

Example syntax:

body { 
	background-color: lightblue;
}

h1 {
	color: white;
	text-align: center
}

p {
	font-family: verdana
	font-size: 20px
}

               Property    Value     Property
